# Solar Tracking & Cleaning System

This project contains the firmware and web dashboard for an advanced Dual-Axis Solar Tracking System with automated cleaning capabilities.

## Structure

*   **/arduino**: Arduino Firmware (`.ino`) for the tracking and cleaning logic.
*   **/web_dashboard**: React/Vite web application for control and monitoring.
*   **SOLAR_TRACKING_SYSTEM_DOCUMENTATION.md**: Detailed system documentation.

## Getting Started

1.  **Firmware:** Open `arduino/Board.ino` in Arduino IDE and upload to your board.
2.  **Dashboard:**
    *   `cd web_dashboard`
    *   `npm install`
    *   `npm run dev`

See [SOLAR_TRACKING_SYSTEM_DOCUMENTATION.md](./SOLAR_TRACKING_SYSTEM_DOCUMENTATION.md) for full details.
